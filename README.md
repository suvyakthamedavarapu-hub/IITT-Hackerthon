
# 🧠 Drone Image Feature Extraction using AI/ML

## 📌 Overview

This project focuses on extracting and identifying features such as buildings, roads, and water bodies from images using a trained AI/ML model.

---

## 🛠️ Technologies

* Python
* TensorFlow / Keras
* NumPy
* OpenCV

---

## 📂 Files

* `mainapp.py` – Model training
* `trained_model.h5` – Saved model
* `test_model.py` – Execution file
* `sample image` – Input

---

## 🚀 Steps

1. Collected and prepared image dataset
2. Preprocessed images (resize, normalize)
3. Built and trained model
4. Evaluated performance
5. Saved trained model
6. Created execution file (`test_model.py`)
7. Loaded model and provided sample image
8. Predicted image features
9. Displayed output

---

## ▶️ How to Run

python mainapp.py  
python test_model.py

---

## 📊 Output

Displays predicted feature from image (e.g., Building, Road, Water)

---

## 💡 Conclusion

The model demonstrates how AI can be used to extract meaningful features from images for applications like land mapping and planning.
